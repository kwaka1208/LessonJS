# Lesson3 サインを作る

## 概要
以下の動作になるように、"MessageBoard()"と"updateMessage()"関数を作成してください。

1. "ただいまキャンペーン実施中！！！"の一文字目だけを表示します。
1. 一秒後には、さらにもう一文字多く表示し、"ただ"まで表示します。
1. この処理を繰り返して、表示内容が一文字ずつ増えていくようにし、全文字が表示されたら処理を終了してください。
